# WGAN-GP
pytorch implementation of wgan-gp

# Usage
- `python3 train.py <exp_name>`
- Config file must be provided at `./configs/<exp_name>.py`
- experiment logs saved in `./EXP_LOGS/log_<exp_name>.txt`
